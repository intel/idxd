
NAME
====

accel-config-disable-device - disables an accfg device

SYNOPSIS
========

>     accel-config disable-device <device>

EXAMPLE
=======

accel-config disable-device dsa0

OPTIONS
=======
-f::
--force::
	Force the disable of device even when there are clients using the
	device.
